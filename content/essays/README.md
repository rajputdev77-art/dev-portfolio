# Essays

Drop markdown files here to add essays to the site.

## Format

```markdown
---
title: "Your Essay Title"
slug: "your-essay-slug"
date: "2026-04-08"
description: "One-line description for the card on the home page."
order: 1
---

Your essay content here in markdown.
```

The slug must match the filename (without .md). Essays appear on the home page in the "How I Think" section and are accessible at `/essays/[slug]`.
